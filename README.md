# Netflix-Data-Cleaning-and-Analysis
This project demonstrates a complete data pipeline using Python and SQL server to ingest, clean, transform and analye Netflix data.
The raw data contained inconsistenciea such as duplicate records, missing values and multi-vlaued columns. This project focused on converting raw data into a structured format and extracting meaningful insights. 

Tech Stack: Python, pandas, SQLAlchemy, SQL Server, Jupyter Notebook

#Data Pipeline Architecture:
CSV Dataset -> Jupyter Notebook (python) -> SQL server -> Data Cleaning -> Transformation -> Analysis

# Key Analysis Performed
1. Number of movies and TV shows created by each director
2. Country with the highest number of comedy movies
3. Top directors per year based on content added
4. Average duration of movies by genre
5. Directors who created both horror and comedy movies

Dataset: Netflix Movies and TV Shows (Kaggle)
